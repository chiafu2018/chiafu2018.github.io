# 課程內容

此課程包含硬體到軟體，當然也包含如何讓程式平行運算，減少執行時間。
以下是上課的五大課程主題

  1. C/C++ Threading
  2. OpenMP
  3. SIMD Programming
  4. Message Passing Interface
  5. GPUs

# 上課方式

一週兩次，一次上課，一次lab。上課的部分就和台灣的上課模式一樣，至於lab部分有課堂練習和回家作業。課堂練習會有助教教你，回家作業則是課堂練習的變形。難度由簡單到難，我自己是完成作業一到六，之後就沒有心和餘力去寫了XD。課程有錄影，所以可以看影片，也可以去實體。因為我住的地方離教室需要2小時的通勤時間，所以我都採取看錄影帶的方式上課。

# 考試作業

八次作業，一次期末考。作業只有加分作用，八個作業只須完成六個就可以在期末的時後讓總成績上升一個級距(總成績減0.3)。期末考試一次定生死(是有補考機制，但要等到下個學期的開學)，所以期末考的時候要全力以赴，因為課程份量有點多。平時寫作業也對期末考試很有幫助。如果是大神的話，可以不寫作業，一樣可以拿滿分!

# 心得

我推薦想要學習平行程式設計的同學修這堂課，不過因為只看期末考的成績決定是否有通過，所以期末要狂讀書，我自己差點沒有通過這門考試，所以要有心理準備。會有考古題讓同學回去練習(統一由老師給)，不過沒有解答，必須自行查答案。期末考試和考古題類似，不過請盡量以他授課的內容去回答，不然分數會被扣光(有著慘痛的經驗)。
