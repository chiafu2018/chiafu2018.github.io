# 課程內容

教你如何和其他人共同開發專案。課程中你會熟悉使用git，和在開發專案所需要的技能。我個人覺得這些技能非常重要，可是在陽明交通大學我沒有找到課程內容類似的課程，讓我覺得很可惜。(可能學校想說要多教一些專業知識，這些技能進入職場在培養就可以)
以下是上課的五大課程主題

  1. Build System
  2. Unit Test
  3. Test Driven Development
  4. Development Tools
  5. Object Oriented Programming

#上課方式

兩週上一次實體課，沒有課的那個禮拜做lab。lab兩人一組，我是和我一起交換的江廷威學長一組。

# 考試作業

五次作業，一次期末考。作業佔總百分比50%，考試佔50%。作業繳交的方式是以gitlab繳交，每兩個禮拜一次作業。作業和考試難度算簡單，很適合交換的同學修課。不過是大學部的課程，研究所的同學應該是不能選。

# 心得

因為我沒有實習經驗，所以我剛開始不太會使用git版本開發工具。所以我在這堂課學習到許多很基本卻很有用的知識。對於日後的軟體開發過程有有更進一步的了解，也謝謝江廷威學長，幫我解決了許多困難。
